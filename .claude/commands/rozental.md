Используй скил `rozental`. Если в запросе явно сказано «только ошибки» / «корректорский
разбор» / «что не так» — работай в режиме Разбор (список флагов с § правил, текст не правишь).
Иначе — Правка: пройди по проходам (типографика → орфография → пунктуация → согласование/
управление → единообразие → самопроверка), отдай таблицу флагов с обязательной колонкой
«Правило», раздел «На решение автора» для спорного и вычитанный текст.

Главное правило: precision важнее recall. Гиперкоррекция (исправление верного на неверное)
для корректора хуже пропуска. Не можешь назвать § правила — не правь.
